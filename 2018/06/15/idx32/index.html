使用vue-cli脚手架一键搭建工程

发表于 2018-06-15 | 阅读数

安装淘宝npm镜像，提高包下载速率

npm install -g cnpm –registry=https://registry.npm.taobao.org

搭建vue工程五步走

1.全局安装vue-cli

npm install -g vue-cli

2.使用一下命令初始化一个项目

vue init webpack my-project

这个命令会在当前目录下创建一个my-project的目录作为工程目录
3.进入项目

cd my-project

4.安装依赖

npm install

5.启动项目

npm run dev
